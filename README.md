韩剧TV - 开源的韩剧影视站点解决方案

欢迎来到 韩剧TV 项目！这是一个基于 Vue3 和 TypeScript 构建的 开源韩剧影视网站解决方案。我们专注于为 韩剧爱好者 提供最新、最全的 高清韩剧在线观看 体验，支持 韩剧资源接口自定义，并且具备 无广告极速播放 功能。

项目特点

海量韩剧资源：包含最新的韩剧、韩剧全集、韩国综艺节目等，支持一键搜索和快速播放。

无广告极速播放：纯净播放体验，告别广告困扰。

每日更新：实时同步最新的韩剧内容，确保第一时间观看到最新剧集。

高清全集：提供高清晰度的韩剧全集在线播放，无缓冲不卡顿。

模块化设计：支持自定义接口，扩展功能方便，适合开发者贡献。

TypeScript类型安全：全面支持TypeScript，增强代码可维护性和安全性。

响应式设计：无论是在手机、平板还是电脑，均能获得最佳的观看体验。

技术栈

Vue3：现代化的前端框架，性能优化，开发体验友好。

TypeScript：类型安全，减少运行时错误，增强代码可读性和可维护性。

Vite：快速的构建工具，提升开发效率和生产环境构建速度。

Pinia：轻量级的状态管理，简化数据管理。

Vue Router：前端路由管理，支持SPA（单页应用）切换。

Element Plus：基于 Vue3 的 UI 组件库，快速构建现代化界面。

安装与使用

克隆项目

git clone https://github.com/your-repo/hanjudy.git


安装依赖

cd hanjudy
npm install


启动开发服务器

npm run dev


构建生产版本

npm run build
